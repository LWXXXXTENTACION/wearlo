# Wearlo

Wearlo 是一个面向年轻人的 AI 穿搭购物平台，提供商品浏览、整套搭配、个性化推荐、智能导购和电商交易能力。

## 技术方向

- Web：Next.js、React、TypeScript
- 商城核心：Go
- 智能电商助手：Python、FastAPI
- 数据：PostgreSQL、pgvector、Redis
- 基础设施：对象存储、Docker、CI/CD
